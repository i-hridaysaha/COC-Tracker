Paste your village JSON

Your current defense, resource-building, trap and wall levels. Saved in this browser only — it won't show up on your other devices. For that, use the "Add/Edit on GitHub" button on the Overview page instead, which commits the file to the repo so every device picks it up. Names must match the game exactly (case doesn't matter) — "buildings" or "defenses" both work as the key for defensive buildings. A raw account export with numeric building ids (e.g. from a base-analysis tool) works too, translated automatically.
